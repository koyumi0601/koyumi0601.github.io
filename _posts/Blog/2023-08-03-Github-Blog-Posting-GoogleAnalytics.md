# Work Instruction

- Go to _config.yml

```yml
# Analytics
analytics:
  provider               : "google-gtag" # false (default), "google", "google-universal", "google-gtag", "custom"
  google:
    tracking_id          : "your id generated from google analytics"
    anonymize_ip         : false # true, false (default)
```
