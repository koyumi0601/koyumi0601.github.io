*.sh file to run script automatically*





```bash 
sudo chmod +x install_script.sh # 실행권한 할당
sudo yes | ./install_script.sh # insert y automatically for every installation questions
```

크롬 설치 중, 클릭, 창 끄기 수작업으로 해야 함
