*DVD backup PC in Ubuntu*

- install Handbrake

```bash
sudo apt update
sudo apt install handbrake
```

- DVD decoding

```bash
sudo apt update
sudo apt install libdvd-pkg
sudo dpkg-reconfigure libdvd-pkg
```
