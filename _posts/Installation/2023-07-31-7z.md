*File compression program*

# Work Instruction

# Install 7 zip
```bash
sudo apt-get update
sudo apt-get install p7zip-full
7z x filename.7z
```

# Unzip

```bash
7z x filename.zip
```
# Unzip in folder name
```bash
7z x filename.zip -ofoldername
```



- for Window, 7z download: [https://www.7-zip.org/download.html](https://www.7-zip.org/download.html)
