*DVD media player in Ubuntu*

- install VLC

```bash
sudo apt install vlc
```



- .wav -> .mp3

  - media > convert/save

  - 각 파일 속성 > 자세히 > 음악가, 앨범 정보 기입

    ```bash
    sudo apt install id3v2 # install id3v2
    id3v2 --album "새 앨범 이름" *.mp3
    id3v2 --artist "Clara Haskill; Schumann" 슈만_트로이메라이.mp3
    id3v2 --TYER "새로운 년도" *.mp3
    ```
